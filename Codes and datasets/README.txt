Emotional_Detection

-> "results_tweetwise.py" gives the experiment results 
-> Files starting with "testing_" are the actual codes to be executed and they use modules ( explained below ) 
-> all other files with extension .py are the actual modules 
-> "datasets" folder contains different datasets in .csv format 
-> "models" contains machine learning models objects saved in format 
-> "utils" contains important utitlity files required to execute codes 
-> "requirements.txt" contains modules required to execute programs

create a python virtual environment ( https://docs.python.org/3/library/venv.html ) and install requirements from requirements.txt using the following command
python -m pip install -r requirements.txt
